#O QUE É O SISTEMA?
    *O sistema que vou desenvolver, é um site de venda e conserto de instrumentos musicais, focados para clientes que desejam adquirir, revender, trocar, consertar ou modificar seus instrumentos, e também voltado para os luthiers, onde além de ter essas funcionalidades listadas acima, terá lista de instrumentos para serem trabalhados, materiais para realizar os concertos, chamados para realizar trabalhos de acordo com especialidade do luthier e controle do andamentos da manutenção dos instrumentos.

#FERRAMENTAS UTILIZADAS:
    *Vou usar todos os conhecimentos que aprendemos até agora sobre o Angular.
